## 下载 PEAR
```
$ brew install autoconf

$ curl -O http://pear.php.net/go-pear.phar
$ [sudo] php -d detect_unicode=0 go-pear.phar
```
<!-- more -->

## 配置和安装 PEAR
1. 选择 1 然后按回车键。      
2. 输入:
`/usr/local/pear`   
3. 选择 4 然后按回车键。    
4. 输入
`/usr/local/bin`   
5. 接下来一直按回车键。
